# Task 13: Demo Files

## Goal

Provide example `.xsp` scenes for development and acceptance testing.

## Scope

- Add a minimal pendulum scene.
- Add a simple spring chain.
- Add a small mesh or jello-like grid.
- Keep files human-readable and deterministic.

## Acceptance Notes

- Demo files load without parser errors.
- Demo files are small enough to understand and edit by hand.

## Done When

- Acceptance or unit tests load every demo file.
- At least one demo file exercises fixed masses and one exercises multiple springs.
